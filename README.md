# TerminalTyper

TerminalTyper is a Textual-based terminal typing test with word-count mode, timed mode, persistent settings, personal bests, and run history.

See [docs/README.md](docs/README.md) for installation, runtime, storage, and development details.

Current release: `0.1.0`.
